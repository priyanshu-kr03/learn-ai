6 to 12 month path

Study plan.

Learn in cycles: read the topic, run a tiny example, break it intentionally, fix it, and write a short explanation in your own words.

Fast 6-month path

  • Month 1: Python, setup, NumPy, Pandas, visualization.
  • Month 2: linear algebra intuition, calculus intuition, probability, statistics.
  • Month 3: ML foundations, data preparation, regression, classification, evaluation.
  • Month 4: ensembles, clustering, feature engineering, recommendation, time series.
  • Month 5: deep learning, PyTorch, CNNs, transformers, NLP.
  • Month 6: LLMs, prompts, embeddings, RAG, agents, deployment, portfolio polish.

Deeper 12-month path

  • Spend two weeks on every major project and keep a public README for each one.
  • Repeat evaluation topics across classic ML, deep learning, and LLM apps.
  • Add MLOps, security, monitoring, system design, and a capstone in the final quarter.
